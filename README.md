# Treasury & Banking Audit Analytics

Enterprise-grade Streamlit dashboard for treasury and banking audit intelligence, built by **CAP AI**.

## Features

- **Bank Reconciliation Aging** — unmatched items, aging buckets, long-pending alerts
- **Round Tripping Detection** — circular transfers, repeated amounts, network graph
- **Bank Charges & Interest Validation** — excess/undercharge flagging
- **Idle Balance Monitoring** — missed earnings estimation
- **Authorization Verification** — unauthorized and duplicate approvals
- **Additional Analytics** — cash flow, spending, vendors, accounts, heatmaps, Sankey diagrams
- **Excel/CSV Import & Export** with auto sheet detection
- **AI-generated audit remarks** and composite risk scoring

## Quick Start

cd C:\Users\Admin\Projects\treasury-audit-analytics
